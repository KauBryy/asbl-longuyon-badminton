# AS Badminton Longuyon

Bienvenue sur le site officiel de l'AS Badminton Longuyon, un club convivial et passionné !

## À propos du club

Nous accueillons des joueurs de tous niveaux, du débutant au compétiteur, dans une ambiance familiale.
Rejoignez-nous au **Gymnase Jean Bouin** !

## Fonctionnalités du site

- **Actualités** : Les dernières nouvelles du club.
- **Équipe** : Présentation des membres et du bureau.
- **Contact** : Formulaire pour nous joindre.
- **Informations pratiques** : Horaires, tarifs et localisation.

## Installation et Développement

Pour lancer le projet localement :

1.  Cloner le dépôt :
    ```sh
    git clone <VOTRE_URL_GIT>
    cd asbl-longuyon-badminton
    ```

2.  Installer les dépendances :
    ```sh
    npm install
    ```

3.  Lancer le serveur de développement :
    ```sh
    npm run dev
    ```

## Technologies utilisées

- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI

## Déploiement

Le site est déployé via GitHub Pages.
Pour déployer une nouvelle version :
```sh
npm run deploy
```
